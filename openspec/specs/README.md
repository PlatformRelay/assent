# Specs

Accepted, living specifications — one directory per capability/epic (E1..E9, see
[meta-plan Phase 5](../../docs/planning/meta-plan.md)). Populated during meta-plan Phase 3
(contracts first). Empty by design until then.

Every behavioural requirement: Given/When/Then, stable REQ ID (`REQ-E1-S01-01`), `Test:`
artifact path, `Verify:` command, and a level tag (L0–L3 per
[ADR-0006](../../docs/adr/0006-testing-strategy.md)).
